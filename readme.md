# 新興救護分隊出勤管理系統

## 系統簡介

本系統為「新興救護分隊出勤管理系統」，

主要用途為：

- 協勤簽到
- 協勤簽退
- 公差勤務紀錄
- 常年訓練簽到
- 常年訓練請假
- 協勤時數統計
- 出勤紀錄查詢
- 資料匯出

系統採用：

- Bootstrap 5
- jQuery
- DataTables
- Google Apps Script
- Google Sheet

建構，

以簡單、穩定、易維護為主要設計方向。


---

# 系統功能

## 1. 使用者切換

可切換：

- 單位
- 職稱
- 姓名

目前登入資訊會顯示於 Navbar 上方。


---

## 2. GPS 定位檢查

系統可依照設定地點進行定位判斷。

目前支援：

- 新興分隊
- 日月光 K11
- 吉林街

若超出允許範圍：

- 將無法進行部分簽到簽退操作。

定位失敗時：

- 可重新定位
- 可確認瀏覽器定位權限
- 可開啟 WiFi 或藍牙提升定位精度


---

# 3. 簽到功能

支援：

- 協勤
- 常年訓練
- 公差勤務

常年訓練：

- 每月僅允許一次
- 可選擇：
  - 簽到
  - 請假

送出前：

- 系統會顯示確認視窗。


---

# 4. 簽退功能

支援：

- 協勤
- 公差勤務

功能包含：

- 工作內容填寫
- Canvas 手寫簽名
- 自動計算時數

送出前：

- 系統會顯示確認視窗。


---

# 5. Summary 統計區

首頁提供：

## 總協勤時數

顯示：

- 累積協勤總時數

---

## 本月協勤時數

顯示：

- 本月時數
- 進度條
- 每月 4 小時目標

若未達：

- 顯示近三個月累積時數。

---

## 常年訓練次數

顯示：

- 年度完成次數
- 12 次目標進度條


---

# 6. 出勤紀錄查詢

支援：

- 全部資料
- 月份查詢
- 日期區間查詢

使用：

- DataTables Responsive

支援：

- 手機版
- 搜尋
- 分頁


---

# 7. 資料匯出

系統提供：

- 出勤紀錄資料匯出功能

可依照條件：

- 匯出 Excel
- 匯出 CSV
